## 2. 范围与风险

### 变更摘要

- **品牌识别边缘误判修复（#620）**：基于 07-20/21 两天全量审计（264 个状态变更、114 条人工裁决、加权精确率 ≈93%）的两处确定性修复——渠道缩写/昵称自介误命中英数别名（「我是BS上加的」「我是🥚冠Bing」）、同轮又要又不要误清无关在位品牌（「肯德基年龄不行」清掉在位六姐）。
- **地理域 B 批（#628/#629）**：区级锚点距离一律渲染估算口径并确定性传递精度；geocode 已解析出城市/坐标时不再反问城市或误报未找到。
- **地理域基线与规格（#627/#630）**：改造方案文档定稿 v3.1；Phase 0 行为基线纯测试固化（候选排序/地理映射/高置信提取/门店定位）。
- **发版链自愈（#631）**：元数据步骤入口重建 git 凭证，修复 claude-code-action 污染凭证导致的 fetch 400 全链断档（07-21 10:22 起五个 PR 元数据断档，本批已全部补录）。

### 高风险区域

- **P1：品牌解析**——#620 属误判收敛（黑名单降档 + 自介守卫扩展 + reducer 净否定闸门），失败模式是过度抑制（真实品牌意向被吞）；全名精确匹配主干不受影响。
- **P1：岗位距离披露（#628/#629）**——geocode 城市披露与估算口径为工具层文案变化，可能影响模型措辞；行为由单测 + Phase 0 基线锁定。
- **P2：发版链（#631）**——仅 workflow shell，不触及业务运行时。
