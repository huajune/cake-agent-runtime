## 2. 范围与风险

### 变更摘要

- **模型配置与桌面布局（#657）**：配置页展示各 Agent 角色实际生效模型、配置来源、覆盖状态与待保存项；补充模型能力和发布日期，并将保存快捷键调整为 Cmd/Ctrl+S、侧栏切换调整为 Cmd/Ctrl+B。
- **聊天图片原图（#657）**：聊天记录优先直接加载原图，原图失败时只回退一次缩略图，同时保留点击放大和新窗口查看能力。
- **Geo 目录收敛（#658）**：`resolution/geo` 与测试目录改为平铺结构；仅移动文件并改写内部相对导入，外部仍通过 `@resolution/geo` 门面访问，运行行为不变。
- **品牌状态单一来源（#662）**：品牌解析、记忆、会话与 Agent 上下文统一收口到 `brand_state`；停止向 `preferences.brands` 回写品牌，并保留兼容读取边界。
- **抽取示例身份回声防护（#662）**：事实抽取结果落库前拦截提示词示例中的姓名、手机号和工作经历组合，避免无关候选人被污染为示例身份。

### 高风险区域

- **P1：模型配置可视化与保存**——实际模型解析优先级展示错误可能误导运营，快捷键冲突可能导致未保存；后端门面测试、前端构建和全量 CI 锁定接口与交互。
- **P1：聊天图片加载**——原图 URL 不可用时可能出现空白或重复请求；组件仅回退一次缩略图，并保留原有放大入口。
- **P2：Geo 物理目录移动**——遗漏内部导入会导致构建或运行时模块解析失败；15 个定向套件、类型检查、构建与全量 CI 已覆盖。
- **P1：品牌与候选人记忆**——旧消费者若继续依赖 `preferences.brands` 可能丢失品牌上下文；统一门面、兼容读取和 93 条定向测试覆盖状态迁移边界。
- **P1：事实抽取防误写**——过滤过宽可能误删候选人真实信息；仅在提示词示例身份组合命中时清理，并以两条线上来源 badcase 完成真实链路回放。
